# Training Planner AI

Персональный рекомендатель тренировок на базе большой языковой модели (GigaChat API).  
Пользователь заполняет профиль, а система генерирует сбалансированную программу тренировок на неделю с учётом целей, уровня подготовки, доступного оборудования и медицинских ограничений.

**Статус:** учебный проект (курсовая работа)  
**Университет:** РЭУ им. Г. В. Плеханова, ВШКМиС, кафедра информатики  
**Дисциплина:** Информационный менеджмент  
**Автор:** Нетиевский Иван Сергеевич  
**Стек:** Python, FastAPI, GigaChat API, HTML/CSS, JavaScript  

## Что умеет система

- Принимает от пользователя: цель, уровень подготовки, количество тренировок в неделю, длительность занятий, доступное оборудование, медицинские ограничения и пожелания.
- Формирует промпт для LLM с ролью «профессиональный тренер».
- Получает от GigaChat структурированный план тренировок (JSON).
- Отображает план в удобном веб-интерфейсе с разделением по дням, упражнениям и методическими комментариями.

## Как запустить локально

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/I1ne/training-planner-ai.git
   cd training-planner-ai
    ```
    
2. Создайте и активируйте виртуальное окружение:
   ```bash
python -m venv venv
source venv/bin/activate  # для Windows: venv\Scripts\activate
   ```

3. Установите зависимости:
   ```bash
pip install -r requirements.txt
   ```
   
4. Создайте файл .env в корне проекта и добавьте ваши учётные данные GigaChat API:
   ```bash
text
GIGACHAT_CREDENTIALS=ваш_токен
GIGACHAT_SCOPE=GIGACHAT_API_PERS
GIGACHAT_MODEL=GigaChat
GIGACHAT_CA_BUNDLE_FILE=путь/до/сертификата.pem
   ```

5. Запустите сервер:
   ```bash
python -m uvicorn app:app --reload
   ```
   
6. Откройте в браузере http://127.0.0.1:8000
